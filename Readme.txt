Tahapan atau tata cara untuk run program kami

1. Download file zip
2. extract file
3. Open file python menggunakan IDE seperti visual studio code (jika mempunyai aplikasi IDE pada device)
4. Run program python

jika tidak menggunakan IDE
1. Download file zip
2. extract file
3. Buka terminal
4. masuk ke dalam folder yang anda gunakan untuk menyimpan file program yang sudah di extract
5. gunakan perintah cd "path tempat menyimpan file" untuk beralih ke folder
5. Run program dengan menggunakan perintah python fuzzy.py
